## Hostel Mess Management and Feedback System

## Overview
This project is a DBMS-based web application for managing hostel mess operations and collecting student feedback. It uses a Flask backend with a MySQL database and a frontend built with HTML, CSS, and JavaScript.

## Features
Student and Admin authentication
View daily mess menu
Submit feedback (rating, complaint, suggestion)
Admin dashboard with analytics and filters
Menu management (add, view, delete)

## Tech Stack
Backend: Flask (Python)
Database: MySQL
Frontend: HTML, CSS, JavaScript

## Database Concepts
Relational schema design
CRUD operations
Joins and queries
Data normalization

## Conclusion
Demonstrates practical implementation of DBMS concepts with full-stack integration for real-world application.
